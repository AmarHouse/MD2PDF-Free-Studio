# Contributing

Thanks for considering contributing to MD2PDF Free Studio!

## Ways to Contribute

- **Report bugs** — Open an issue with clear steps to reproduce.
- **Suggest features** — Open an issue describing what you'd like to see.
- **Submit PRs** — Fix bugs, add features, improve documentation.
- **Share** — Star the repo, share on social media, tell other developers.
- **Create themes** — Use the built-in theme editor and share your creations.

## Pull Requests

1. Fork the repo
2. Create a branch: `git checkout -b my-feature`
3. Make your changes
4. Test: `node --check js/your-file.js`
5. Commit and push
6. Open a PR

## Code Style

- No frameworks, no build step — keep it vanilla JS
- Match existing code style and conventions
- Test syntax with `node --check` before committing

## AgenticPDF

If you want to improve the AgenticPDF system for AI agents, focus on:
- `AgenticPDF/agentic.mjs` — core rendering engine
- `AgenticPDF/themes.mjs` — theme data
- `AgenticPDF/config.json` — bilingual configuration

## Questions?

Open a discussion or issue. We're friendly!
